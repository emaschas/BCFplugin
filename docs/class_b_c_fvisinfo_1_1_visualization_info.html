Detailed Description

Visualization Info (.bcfv) definition

The Visualization Info includes the following properties :

CamTypeCamera type (NoCamera, Perspective or Orthogonal)
CamX, CamY, CamZCamera location (in meters)
DirX, DirY, DirZCamera direction
UpX, UpY, UpZCamera Up vector
FieldField of view in degrees
ViewToWorldScaleView to World Scale
ComponentsList of visible components
